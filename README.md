# hospital-db-anomaly-detection-siem
Projet de cybersécurité réalisé au CHU Hassan II à Fès. Développement d’un mini-SIEM en Python et MySQL pour centraliser les logs, détecter automatiquement les accès suspects (hors horaires, requêtes massives, échecs répétés), générer des rapports CSV et envoyer des alertes e-mail via une interface web Flask.
